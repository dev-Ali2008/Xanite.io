General Questions

What is Xanite Emulator?

Xanite is a powerful and optimized emulator for Android devices that allows you to play original Xbox and Xbox 360 games on your mobile device. It's built from the ground up specifically for Android, offering high performance and compatibility.

Key Features:

  • port emulator Android implementation (not Termux-based)
  • Vulkan 1.1 support
  • High compatibility with Xbox and Xbox 360 games
  • Optimized for modern Android devices
  • Controller and touchscreen support
Is Xanite free to use?

Yes! Xanite will offer two versions:

Xanite Free: Complete emulator with all core features for playing Xbox and Xbox 360 games.

Xanite Gold (Premium): Enhanced version with additional features like:

  • Priority updates and early access to new features
  • Advanced graphics settings and enhancements
  • Cloud save synchronization
  • Premium support and priority bug fixes

Both versions will be available through our official channels.

Does Xanite reuse code from other emulators?

No! Xanite is built entirely from scratch specifically for Android. We do not reuse or copy code from other emulators like Xemu, Xenia, or any other projects.

Why this matters: Building from scratch allows us to:

  • Optimize specifically for Android architecture
  • Implement modern graphics APIs efficiently
  • Create a clean, maintainable codebase
  • Add features unique to mobile gaming

Our code is open source and available on GitHub for transparency.

Installation & Setup

How do I install Xanite on Android?

Installation Guide:

Minimum Requirements:

  • Android 9.0 or higher
  • ARMv8 architecture (64-bit)
  • At least RAM 6GB recommended
  • Vulkan 1.1 support

Installation Steps:

  1. Download the latest APK from our download page or GitHub releases
  2. Enable "Install from Unknown Sources" in your Android settings
  3. Locate the downloaded APK file and tap to install
  4. Follow the on-screen installation instructions
  5. Launch Xanite from your app drawer

⚠️ Security Note: Only download Xanite from our official sources. Avoid third-party websites that may distribute modified or malicious versions.

Do I need BIOS files or game files?

Yes, you need game files. Xanite does NOT come with any games or BIOS files. You must provide your own legally-owned game files.

What you need:

  • Game ISO files - Disc images of Xbox 360 games
  • Game ROMs - Digital copies of games
  • BIOS (for Xbox) - Required for Xbox emulation

⚠️ Legal Notice: You must own the original games to legally use their files with Xanite. We do not support or encourage piracy.

How to add games:

  1. Place your game files in a folder on your device
  2. Open Xanite and go to Settings → Game Directories
  3. Add the folder containing your game files
  4. Xanite will automatically scan and add games to your library

Game Compatibility

Which games work with Xanite?

Xanite aims to support a wide range of Xbox 360 games. Compatibility varies based on game requirements and device capabilities.

Current Compatibility Status:

  • Xbox 360: ~2500 games in database, testing in progress
  • Best Working: 2D games, older titles, arcade games
  • Work in Progress: 3D games, AAA titles, newer releases

Check Compatibility: Visit our compatibility page for the complete list of tested games and their status.

Tip: Games marked as "Playable" or "Perfect" in the compatibility list will work best. Games marked as "Nothing" have not been tested yet.

How is compatibility determined?

We use a standardized compatibility rating system similar to other emulators:

Compatibility Ratings:
🟢 Perfect - Game runs flawlessly with no issues
🟡 Playable - Minor issues, fully playable
🔵 In-Game - Major issues, but reaches gameplay
🟠 Intro - Only intro/menu works
🔴 Broken - Crashes or freezes
⚫ Nothing - Won't start or untested

Testing Methodology:

  1. Initial boot test (does it start?)
  2. Menu navigation test
  3. Gameplay test (30+ minutes)
  4. Performance metrics (FPS, stability)
  5. Bug reporting and documentation

Community Testing: You can help! Test games on your device and report results on our Discord or GitHub.

Performance & Optimization

What are the minimum requirements?

Minimum Requirements:

  • Android Version: 9.0 or higher
  • CPU: ARMv8 (64-bit) processor
  • RAM: 6GB recommended
  • GPU: Vulkan 1.1 support
  • Storage: 500mb free space for emulator
  • Additional: 2-4GB per game for game files

Recommended Specifications:

  • Android Version: 10 or higher
  • CPU: Snapdragon 845/855 or equivalent
  • RAM: 8GB or more
  • GPU: Adreno 640 or better
  • Storage: UFS 3.0/3.1 for faster loading

Performance Tips: Close background apps, enable performance mode, and use a cooler for extended gaming sessions.

How can I improve performance?

Performance Optimization Guide:

Quick Performance Boost:

  1. Close all background applications
  2. Enable "Performance Mode" in your device settings
  3. Disable battery optimization for Xanite
  4. Use a cooling fan or play in a cool environment

Xanite Settings Optimization:

  • Graphics Settings:
    • Resolution: Set to 1x or 720p for best performance
    • Anti-aliasing: Disable for performance boost
    • Texture filtering: Use "Bilinear" instead of "Trilinear"
    • Shader compilation: Enable async compilation
  • CPU Settings:
    • CPU mode: Use "Performance" or "Balanced"
    • Recompiler: Enable for better performance
    • Multithreading: Enable if your device has 8+ cores
  • Memory Settings:
    • Memory allocation: Set to "Balanced"
    • Cache size: Increase if you have 8GB+ RAM

⚠️ Warning: Some settings may cause instability. Adjust one setting at a time and test thoroughly.

Troubleshooting

Game crashes or won't start - What to do?

Troubleshooting Steps:

Quick Fix Checklist:
1. Check game compatibility status
2. Verify game file integrity
3. Update Xanite to latest version
4. Clear Xanite cache and data
5. Try different graphics settings
6. Check device meets minimum requirements

Common Issues and Solutions:

  • Game crashes on startup:
    • Check if game is compatible
    • Try different BIOS files (for Xbox 360 games)
    • Verify game file isn't corrupted
  • Black screen or freeze:
    • Wait 1-2 minutes (first-time shader compilation)
    • Disable "Async Shader Compilation"
  • Poor performance/slowdowns:
    • Lower resolution scale
    • Disable anti-aliasing
    • Close background apps
    • Enable performance mode

Getting Help: If problems persist, join our Discord and provide:

  • Device model and Android version
  • Xanite version
  • Game name and region
  • Error message or screenshot

Xanite won't install - What's wrong?

Common Installation Issues:

Installation Error Solutions:

  • "App not installed":
    • Enable "Install from Unknown Sources"
    • Uninstall previous versions first
    • Clear Google Play Store cache
    • Check if APK file is corrupted (re-download)
  • "Parse error":
    • Download APK again (corrupted download)
    • Check Android version compatibility
    • Try downloading from different source
  • "Insufficient storage":
    • Clear device storage (at least 1GB free)
    • Move apps/files to SD card
    • Clear app caches
  • "Device not compatible":
    • Check minimum requirements
    • Verify ARMv8 (64-bit) CPU
    • Update Android version if possible

Step-by-Step Fix:

  1. Go to Settings → Security → Unknown Sources (enable)
  2. Uninstall any existing Xanite versions
  3. Clear Play Store cache (Settings → Apps → Play Store → Clear Cache)
  4. Download fresh APK from official source
  5. Install and restart device if needed
